# Data Source: okta_oauth_authorization_server


Returns OpenID Connect metadata for the Okta org authorization server. Clients use this information to programmatically configure their interactions with Okta


## Links

- [Okta API docs](https://developer.okta.com/docs/api/openapi/okta-management/management/tag/AuthorizationServer/)
- [Provider source](https://github.com/okta/terraform-provider-okta/blob/master/okta/services/idaas/data_source_oauth_auth_server.go)

## Example Usage

```terraform
variable "hostname" {
  type = string
}

data "okta_oauth_authorization_server" "test" {
  base_url = "https://${var.hostname}"
}
```
