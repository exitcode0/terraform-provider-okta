# Resource: okta_ui_schema


Terraform Resource for okta_ui_schema.


## Links

- [Okta API docs](https://developer.okta.com/docs/api/openapi/okta-management/management/tag/UISchema/)

## Example Usage

```terraform
resource "okta_ui_schema" "test" {
  ui_schema {
    button_label = "submit"
    label        = "Sign in"
    type         = "Group"

    elements {
      type  = "Control"
      scope = "#/properties/lastName"
      label = "Last Name"
      options {
        format = "text"
      }
    }
  }
}
```
