# Resource: okta_device_posture_check

Manages a device posture check. Device posture checks use OSQuery to evaluate device compliance and can be referenced in device assurance policies.

## Example Usage

```terraform
resource "okta_device_posture_check" "example" {
  name          = "Disk Encryption Check"
  platform      = "MACOS"
  query         = "SELECT 1 FROM disk_encryption WHERE encrypted = 1;"
  description   = "Verifies that disk encryption is enabled"
  variable_name = "diskEncryptionCheck"
  mapping_type  = "CHECKBOX"

  remediation_settings = {
    message = {
      default_i18n_key = null
      custom_text      = "Please enable FileVault disk encryption on your Mac."
    }
    link = {
      default_url = null
      custom_url  = "https://support.apple.com/en-us/HT204837"
    }
  }
}
```

## Schema

### Required

- `name` (String) Display name of the device posture check.
- `platform` (String) Platform (OS) the check applies to. Valid values: `MACOS`, `WINDOWS`.
- `query` (String) OSQuery SQL string for the device posture check.

### Optional

- `description` (String) Description of the device posture check.
- `variable_name` (String) Unique variable name used to reference this check in device assurance policies. Must contain only alphanumeric characters.
- `mapping_type` (String) How the check is rendered in device assurance policies. Valid values: `CHECKBOX`, `TEXTBOX`.
- `remediation_settings` (Object) Remediation instructions shown to the end user when the device posture check fails. (see [below for nested schema](#nestedatt--remediation_settings))

### Read-Only

- `id` (String) The ID of the device posture check.
- `type` (String) The type of the device posture check. User-created checks are always `CUSTOM`. Okta-provided defaults are `BUILTIN`.
- `created_by` (String) User who created the device posture check.
- `created_date` (String) Timestamp when the device posture check was created.
- `last_updated` (String) Timestamp when the device posture check was last updated.
- `last_updated_by` (String) User who last updated the device posture check.
