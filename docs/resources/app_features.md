# Resource: okta_app_features


Terraform Resource for okta_app_features.


## Links

- [Okta API docs](https://developer.okta.com/docs/api/openapi/okta-management/management/tag/ApplicationFeatures/)

## Related Resources

- [`okta_app_saml`](../resources/app_saml) — SAML application

~> **NOTE:** This resource cannot be deleted via Terraform. Application features are managed by Okta and can only be updated or read.

~> **NOTE:** This resource is only supported with a limited subset of OIN applications. See the [Okta API docs](https://developer.okta.com/docs/api/openapi/okta-management/management/tag/ApplicationFeatures/) for more details.

## Example Usage

```terraform
resource "okta_app_features" "example" {
  app_id = "0oarblaf7hWdLawNg1d7"
  name   = "USER_PROVISIONING"
  capabilities {
    create {
      lifecycle_create {
        status = "ENABLED"
      }
    }
    update {
      lifecycle_deactivate {
        status = "ENABLED"
      }
      profile {
        status = "DISABLED"
      }
      password {
        status = "ENABLED"
        seed   = "RANDOM"
        change = "KEEP_EXISTING"
      }
    }
  }
  status = "ENABLED"
}
```
